# Titanic-
This project focuses on data cleaning and preprocessing of the Titanic dataset, which is a binary classification problem aimed at predicting whether a passenger survived or not. The dataset contains demographic and travel-related information such as age, sex, passenger class, fare, and embarkation port, with the target variable being “Survived.” The main goal of this work is to prepare a clean and structured dataset suitable for machine learning by handling missing values, removing irrelevant features, and ensuring data consistency. Missing values in the Age column were filled using median imputation, while missing values in Embarked were filled using the mode. The Cabin column was dropped due to a high proportion of missing data, and identifier columns such as PassengerId, Name, and Ticket were removed as they do not contribute to prediction. The Sex column was standardized for consistency, and duplicate records were removed. Basic analysis showed that approximately 38% of passengers survived, with survival strongly influenced by gender and passenger class, where females and first-class passengers had higher survival rates. The final output is a clean, machine-learning-ready dataset that can be used for classification models such as logistic regression, decision trees, or random forest to predict survival outcomes.
